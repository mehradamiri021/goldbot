## 2. دسترسی به وب‌سرویس از IP خارجی

### نحوه اتصال:
```bash
# Replace YOUR_SERVER_IP with your actual server IP
http://YOUR_SERVER_IP:5000/
```

### صفحات در دسترس:
- 📊 Dashboard: `http://YOUR_SERVER_IP:5000/dashboard`
- 📈 Charts: `http://YOUR_SERVER_IP:5000/charts`  
- ⚙️ Settings: `http://YOUR_SERVER_IP:5000/settings`
- 📋 Signals: `http://YOUR_SERVER_IP:5000/signals`

### تنظیم IP سرور:
```bash
# در فایل .env اضافه کنید:
SERVER_IP=YOUR_ACTUAL_IP_ADDRESS
```

## 3. قالب‌های گزارش‌دهی

### گزارش روزانه شامل:
- تحلیل پرایس اکشن و ساختار بازار
- محدوده‌های کلیدی (مقاومت/حمایت)
- ساختار بازار (BOS/CHoCH)
- اندیکاتورها (RSI, MACD)
- آمار ۲۴ ساعته
- تحلیل جامع و چشم‌انداز
- اخبار تأثیرگذار

### گزارش هفتگی شامل:
- تحلیل تکنیکال چند تایم‌فریم
- سطوح مقاومت و حمایت هفتگی/روزانه/۴ساعته
- استراتژی‌های ورود و خروج
- تقویم اقتصادی هفته
