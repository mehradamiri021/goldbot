# 🗄️ راهنمای دیتابیس - GoldBot

## ✅ دیتابیس خودکار تولید می‌شود

**نیازی به نصب دستی PostgreSQL نیست!**

### 🔧 نحوه کار:
1. **Flask خودکار**: دیتابیس را با اولین اجرا می‌سازد
2. **Models.py**: تمام جداول خودکار ایجاد می‌شوند
3. **SQLAlchemy**: مدیریت کامل دیتابیس

### 📊 جداول سیستم:
- `signals` - سیگنال‌های معاملاتی
- `analysis` - تحلیل‌های بازار
- `telegram_messages` - پیام‌های ارسالی
- `bot_status` - وضعیت کامپوننت‌ها
- `price_alerts` - هشدارهای قیمت
- `bot_statistics` - آمار عملکرد

### 🚀 راه‌اندازی:
```bash
# فقط اجرا کنید - دیتابیس خودکار می‌سازد
python3 main.py
```

### 🔍 مونیتورینگ:
- پنل ادمین: http://localhost:5000/admin
- وضعیت دیتابیس: خودکار بررسی می‌شود
- بک‌آپ: خودکار هر روز

## ⚠️ نکات مهم:
- **PostgreSQL**: محیط production از DATABASE_URL استفاده می‌کند
- **SQLite**: محیط development (فایل محلی)
- **Migration**: خودکار با هر تغییر model

✅ **نتیجه**: شما فقط سیستم را اجرا کنید - بقیه خودکار است!
